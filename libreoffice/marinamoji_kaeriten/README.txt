marinaMoji Kaeriten for LibreOffice Writer

INSTALL (macOS — recommended):

1) Download marinamoji-kaeriten-libreoffice-mac.dmg from GitHub Releases.

2) Right-click → Open the installer app if macOS warns.

3) Quit Writer, run "Install marinaMoji Kaeriten (LibreOffice)".

4) Accept the extension in Extension Manager, restart Writer.

5) View → Toolbars → marinaMoji.

INSTALL (all platforms — .oxt only):

1) Extension Manager → Add MarinaMojiKaeriten.oxt → restart Writer.

2) On macOS LibreOffice 26.x, if toolbar buttons do nothing, use the Mac
   installer above (copies Python macros into your LibreOffice profile).

Use the marinaMoji toolbar only (View → Toolbars → marinaMoji).
Five buttons: Render, Unrender, Copy plain text, Copy TEI, Copy LaTeX.
Type 說㆒㆑者, then Render kaeriten (selection if highlighted, else document).

Do not edit inside frames; use Unrender kaeriten to edit ㆒㆑ marks.
