marinaMoji Kaeriten for LibreOffice Writer

INSTALL:

1) Run build.sh in the libreoffice folder.

2) Extension Manager → Add MarinaMojiKaeriten.oxt → restart Writer.

(Python macros are inside the .oxt — required on macOS.)

Use the marinaMoji toolbar only (View → Toolbars → marinaMoji).
Five buttons: Render, Unrender, Copy plain text, Copy TEI, Copy LaTeX (all copies go to the clipboard).
Type 說㆒㆑者, then Render kaeriten (selection if highlighted, else document).

Do not edit inside frames; use Unrender kaeriten to edit ㆒㆑ marks.
